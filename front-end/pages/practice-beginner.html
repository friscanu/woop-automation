Linux Command Line Basics

Beginner Hands-On
Estimated Time: 45 minutes
Tasks: 5

Tasks

  • 1 Navigating the File System

    Learn how to navigate through the Linux file system using the following commands:

    • pwd - Print working directory
    • ls - List directory contents
    • cd - Change directory

    Try the following:

    1. Use pwd to see your current directory
    2. Use ls to list files in the current directory
    3. Use cd /tmp to change to the temporary directory
    4. Use ls -la to show all files including hidden ones

Lab Instructions

This lab will help you practice essential Linux commands that every IT professional should know. Each task focuses on a different aspect of the Linux command line interface.

Getting Started

  1. Work through the tasks in order, using the Linux terminal to the right.
  2. Read each task description carefully before attempting the commands.
  3. Use the hints tab if you get stuck on a particular command.
  4. Use the notes tab to keep track of important commands or concepts.

Completion Requirements

To successfully complete this lab, you must finish all 5 tasks. Each task will be marked as complete when you've successfully executed the required commands.
